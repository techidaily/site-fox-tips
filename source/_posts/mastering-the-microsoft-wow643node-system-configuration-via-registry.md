## Wow6432Node Registry Key

The Wow6432Node registry entry indicates that you are running a 64-bit Windows version.

 The operating system uses this key to display a separate view ofHKEY\_LOCAL\_MACHINE\\SOFTWARE for 32-bit applications that run on 64-bit Windows versions. When a 32-bit application writes or reads a value under theHKEY\_LOCAL\_MACHINE\\SOFTWARE\\<company>\\<product> subkey, the application reads from theHKEY\_LOCAL\_MACHINE\\SOFTWARE\\Wow6432Node\\<company>\\<product> subkey.
