## New Java Virtual Machine System Property Dialog

 In this dialog, you can specify a new Java Virtual Machine system property parameter.

![New Java Virtual Machine System Parameter Dialog](https://cdn.advancedinstaller.com/img/dialog/new-jvm-system-property.png "New Java Virtual Machine System Parameter Dialog")  

Edit the Name/Value fields with the desired values. No special formatting rules are required. Enter the strings in the form you want them to appear as command line parameters. Both fields are of [Formatted Type](https://tools.techidaily.com/advancedinstaller/products/) and can be edited using [Smart Edit Control](https://tools.techidaily.com/advancedinstaller/products/).
